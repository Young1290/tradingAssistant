# 🟢 BTC 宏观与情景分析 AI 提示词模版

## 指令

"请根据以下提供的【分析维度】数据，对比【情景触发条件】，判断当前 BTC 市场最符合哪一种情景，并给出应对策略建议。"

---

## 第一部分：当前实时数据输入

| 分析维度 | 子项 / 指标 | 当前实时数值/趋势 |
|---------|------------|------------------|
| **1. 宏观经济** | 美元指数 (DXY) | *(例如: 98.0, 走弱)* |
| | Fed 利率政策 | *(例如: 降息 25bp, 维持现状)* |
| **2. 资金流动** | BTC ETF 净流入 | *(例如: 单周流入 $1.2B / 单月流出 $3B)* |
| | 长期持有者行为 | *(例如: 30天抛售 45万枚, 抛压减缓)* |
| **3. 市场支撑** | 挖矿生产成本 | *(例如: $94,000)* |
| | 美股表现 (S&P500) | *(例如: 创历史新高 / 出现 5% 回撤)* |
| **4. 风险事件** | 避险需求 / 黑天鹅 | *(例如: 无 / 某交易所爆雷 / 地缘政治紧张)* |

## 第二部分：情景判定逻辑参考表

**请 AI 根据输入数据，匹配以下四个情景之一：**

| 维度 | 情景 1: V型反转 | 情景 2: 高位横盘 | 情景 3: 缓慢熊市 | 情景 4: 深度熊市 |
|------|----------------|----------------|----------------|----------------|
| **Fed 政策** | 开启 QE (量化宽松) | 仅降息，不 QE | 维持现有利率不变 | 衰退 + 政策失败 |
| **持有者行为** | 停止抛售 (<40万枚) | 抛售放缓 (50-60万枚) | 抛售加速 (>100万枚) | 恐慌性抛售 (无底线) |
| **ETF 资金流** | 单周流入 >$1B | 每日小幅波动 (±$50M) | 单月流出 >$2B | 单月流出 >$5B |
| **逻辑支撑** | 信用对冲 + 机构抢筹 | 收益竞争 + 矿工挺价 | 预期透支 + 老玩家离场 | 系统性风险 + 泡沫破灭 |
| **关键技术位** | $100K 守住并 V 反 | $94K 多次测试并守住 | $94K/$90K 接连失守 | $85K 失守 → $50K |
| **美股关联** | 美股爆涨 | 美股走平或微涨 | 美股下跌 | 美股泡沫破灭 |

## 第三部分：应对策略模版 (由 AI 生成输出)

### 输出格式示例

```json
{
  "当前判定情景": "[情景 1 / 2 / 3 / 4]",
  "价格目标预期": "[例如: $120K - $150K / $85K - $90K 等]",
  "操作建议": {
    "仓位管理": "[例如: 分批建仓 / 立即清仓 / 网格交易]",
    "止损/抄底位": "[例如: 设置 $92K 止损 / 等待 $50K 抄底]",
    "风险提示": "[针对当前情景的风险警告]"
  },
  "详细分析": "[详细说明当前符合哪个情景的理由]"
}
```

---

## 使用说明

1. **数据收集**：填写第一部分表格中的实时数据
2. **情景匹配**：AI 对比第二部分的情景条件，找出最符合的情景
3. **策略输出**：AI 根据判定的情景，生成第三部分的操作建议

| 情景 | 核心特征 | 预期价格走势 | 操作策略 |
